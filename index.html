PROJECTS

Smoke detector using Arduino

The project aims to detect smoke typically as an indication of fire. An MQ2 gas sensor is used to sense smoke and sensor output is connected to the Arduino UNO (microcontroller) which alerts the user by controlling to the LED and buzzer based on the sensor output. It can be used as a basic fire alerting device in domestic and small industrial areas.

Transformer fault prediction using Machine Learning Algorithms

The project aims to predict the occurence of fault in a transformer using various outputs like Oil level and temperature indicators from the Transformer and train the model using Logistic regression, Support Vector Machines & Desicion tree classifiers to obtain increasing accuracy of fault prediction. The dataset of input values is taken from Kaggle.

Low-cost Medical ventilator Prototype

This ventilator prototype is implemented using simple DC motor speed control and is hence cost-efficient and can be developed fully. It is a solution to the unmet demand and supply of portable and effective ventilators. A mechanical arm is used to ecert force on the medical bvm so it pumps air under pressure required to aid the breathing requirement of patient.
